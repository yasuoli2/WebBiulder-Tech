🚀 WebBuilder Tech | Landing Page

Projeto de uma landing page moderna e responsiva desenvolvida para uma empresa fictícia de criação de sites e soluções digitais.

A aplicação foi construída utilizando HTML, CSS e Bootstrap , com foco em design atrativo, experiência do usuário (UX) e responsividade em diferentes dispositivos.

✨ Funcionalidades
🔹 Navbar responsiva com navegação suave
🔹 Seção Hero com destaque visual e animações
🔹 Carousel interativo para portfólio
🔹 Seção “Sobre” com apresentação dos serviços
🔹 Accordion com serviços oferecidos
🔹 Formulário de contato estilizado
🔹 Modal de contato
🔹 Botão “Voltar ao topo” com scroll suave
🔹 Ícones e integração com redes sociais
🎨 Destaques do Projeto
Layout moderno com uso de gradientes e animações CSS
Interface totalmente responsiva
Uso de Bootstrap Icons
Estrutura organizada e de fácil manutenção
Experiência visual focada em conversão
🛠️ Tecnologias Utilizadas
HTML
CSS
Bootstrap
JavaScript (básico para interações)
📍 Objetivo

Este projeto foi desenvolvido com fins acadêmicos para praticar conceitos de desenvolvimento front-end, criação de interfaces modernas e utilização de frameworks CSS.
